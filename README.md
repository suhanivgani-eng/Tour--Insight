Tour Insight - Smart Tourism Feedback Analysis

📌 Problem Statement
The tourism industry receives a large amount of feedback from tourists through online reviews and ratings. Manually analyzing this feedback is difficult and time-consuming. Because of this, important insights about tourist satisfaction and service quality may be missed.

 # Objectives
- Automatically analyze tourist reviews using NLP
- Classify feedback into Positive, Negative, and Neutral
- Identify common issues in tourism services
- Provide insights for improving tourist experience
- Visualize feedback using charts and dashboards

# Solution Overview
Tour Insight is an AI-powered system that processes tourist reviews and extracts meaningful insights using Natural Language Processing (NLP) and Machine Learning techniques.


# Features
- ✅ Sentiment Analysis (Positive / Negative / Neutral)
- 📊 Data Visualization Dashboard (Graphs & Charts)
- 🔍 Feedback Analysis for decision-making
- 📈 Insights generation from tourist reviews

# Tech Stack
- **Programming Language:** Python
- **Framework:** Streamlit
- **Libraries:**
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
  - Scikit-learn
  - TextBlob (for NLP)

 System Architecture

1. Data Collection (CSV / Reviews)
2. Data Preprocessing (Cleaning text)
3. Sentiment Analysis (NLP)
4. Data Visualization (Charts)
5. Insights Generation
